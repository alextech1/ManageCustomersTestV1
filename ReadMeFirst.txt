1. Must open SQL Server.
2. Remember Server Name to use on Visual Studio
3. Right click on "Databases" folder, then "Restore Database"
4. Click on "Device", then click "..." to browse and select "ProductsDatabase" that is located in this folder.
5. Then click "OK"
6. Open the solution file.
7. Click on "DataAccess.cs" under "Nile.Windows" project
8. Replace server name on line 28, with your server name.
9. Run the Project!
